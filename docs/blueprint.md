# **App Name**: Al-Melha Community Hub

## Core Features:

- Member Authentication & Profiles: Secure user registration and login system with personalized member profiles and role-based access control (Admin, Member, Committee roles).
- Member Directory: A comprehensive, searchable, and filterable directory displaying registered members' basic public profiles.
- Dynamic Event Calendar & Registration: An interactive calendar for displaying association events, with functionality for members to RSVP or register for participation.
- Community Project Showcase: Dedicated pages to display current and past community projects with detailed descriptions, progress, and relevant information.
- Financial Transparency Portal: A module to present financial reports (e.g., annual budgets, audit reports) and a transparent overview of membership dues collection rates.
- AI-Powered FAQ/Info Chat Tool: An integrated chat tool capable of using generative AI to answer common member queries and provide guided navigation or information.
- News & Announcements Feed: A centralized section for administrators to publish news articles, important announcements, and manage newsletter subscriptions.

## Style Guidelines:

- The primary color, conveying trust and reliability, is a rich blue (#2E6CDC). The background, creating a clean canvas for content, is a desaturated blue (#F2F6FF). A vibrant accent green (#43CC57) symbolizes growth and community vitality, offering visual contrast.
- Headlines and English content utilize 'Poppins' (sans-serif) for its modern and geometric appearance. 'Cairo' (sans-serif) is chosen for all Arabic text, ensuring cultural relevance and excellent readability in RTL layouts. Note: currently only Google Fonts are supported.
- A comprehensive set of modern, crisp icons from Font Awesome will be used across the application to enhance navigation and convey information visually.
- A clean, responsive, and grid-based layout powered by Tailwind CSS. Content is structured into clear, easily digestible sections with support for Arabic (RTL) localization, ensuring a seamless experience across all devices.
- Subtle, non-intrusive animations will be employed for state changes (e.g., loading spinners), modal transitions, and button interactions to enhance user feedback and improve perceived performance.
